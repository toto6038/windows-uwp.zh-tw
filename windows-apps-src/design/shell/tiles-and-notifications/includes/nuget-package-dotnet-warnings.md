> 仍使用 packages.config 的 .NET Framework 桌面應用程式必須遷移至 PackageReference，否則將無法正確參考 Windows 10 Sdk。 在您的專案中，以滑鼠右鍵按一下 [參考]，然後按一下 [將 packages.config 遷移至 PackageReference]。
