# <a name="uri"></a>/{uri}

  * [URI 參數](#ID4EX)
 
<a id="ID4EX"></a>

 
## <a name="uri-parameters"></a>URI 參數
 
| 參數| 類型| 描述| 
| --- | --- | --- | 
| <b>uri</b>| 字串| 要使用的 URI。 請參閱方法頁面，如需詳細資訊。|
